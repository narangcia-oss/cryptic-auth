Expand description

An abstraction for user persistence, allowing async CRUD operations on users.

Implementors of this trait provide mechanisms to add, retrieve, update, and delete users in a persistent store (e.g., database, in-memory, etc). All methods are asynchronous and return results suitable for error handling in authentication contexts.
