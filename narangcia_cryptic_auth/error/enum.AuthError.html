Expand description

All possible errors that can occur within the authentication crate.

This enum is used throughout the crate to represent error conditions that may arise during authentication, user management, password handling, token operations, and more. Each variant is documented to clarify the context in which it is used.

InvalidCredentials

Returned when the provided credentials are invalid (e.g., wrong password or username).

§

UserNotFound

Returned when a user cannot be found in the data store.

§

UserAlreadyExists

Returned when attempting to create a user that already exists.

§

HashingError(String)

Returned when password hashing fails. Contains the underlying error message.

Returned when password verification fails. Contains the underlying error message.

§

ConfigError(String)

Returned when there is a configuration error in the authentication system. Contains a description of the configuration issue.

§

ServiceUnavailable(String)

Returned when a service required for authentication is unavailable. Contains a description of the service issue.

§

NotImplemented(String)

Returned when a feature is not yet implemented. Contains a description of the missing feature.

§

InvalidInput(String)

Returned when input data is invalid or malformed. Contains a description of the invalid input.

§

InvalidPassword(String)

Returned when a password does not meet policy requirements or is otherwise invalid. Contains a description of the password issue.

§

VerificationError(String)

Returned when a verification process fails (e.g., email or token verification). Contains a description of the verification error.

§

TokenExpired

Returned when a token (such as a JWT) has expired and is no longer valid.

§

TokenGeneration(String)

Returned when token generation fails. Contains a description of the generation error.

§

TokenValidation(String)

Returned when token validation fails. Contains a description of the validation error.

§

InvalidToken(String)

Returned when a token is invalid (malformed, tampered, or otherwise unusable). Contains a description of the invalid token.

§

MissingPasswordManager

Returned when the password manager component is missing or unavailable.

MissingPersistentUserManager

MissingTokenManager

Returned when the token manager component is missing or unavailable.

OAuthNetwork(String)

OAuthInvalidResponse(String)

OAuthProvider(String)

OAuthTokenExchange(String)

§

OAuthOther(String)

Returned for other errors related to OAuth operations.

SignupError(String)
