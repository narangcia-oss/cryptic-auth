Expand description

Trait for secure password management, including hashing and verification.

Implement this trait to provide password hashing and verification using a secure algorithm (e.g., Argon2, bcrypt, scrypt). The trait is asynchronous to support non-blocking operations in environments where password hashing may be computationally expensive.
